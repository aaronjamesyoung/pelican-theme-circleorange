The Work

Oregon Travel Experience - Summer/Fall 2011

OTE is a semi-independent state agency that manages Oregon's blue highway signs, several rest areas in the state, and the Heritage Trees and Historical Markers scattered throughout Oregon. This project was part of a large rebranding effort by OTE.

In this project, I developed a design built for OTE by another web agency. During the course of the project, we had to make some minor changes to the design due to a shifting sense of how this website would serve OTE's mission. Other than these changes, the live website is faithful to the original design created for OTE.

I created a custom WordPress theme, building in functionality to help OTE administer the website and increase user engagement. The theme is responsive and search-friendly.
